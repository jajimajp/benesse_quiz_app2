# AOI (Ability Of Imagination)

クイズで楽しみながら思考力を鍛えられるアプリケーション

## 技術スタック

- Flutter
- Firebase

## 環境構築

このリポジトリをクローンします。

```bash
git clone https://github.com/jajimajp/benesse_quiz_app2.git
```

プロジェクトのディレクトリに移動後、アプリの実行に必要な依存環境のインストールを行います。

```bash
flutter pub get
```

デバッグモードで実行します。
```bash
 flutter run --debug
 ```
